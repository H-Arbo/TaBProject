TAB to Stop Development Documentation

Project Name: TAB to Stop

Version: 1.0

Date: April 17, 2024

Overview

  • Educate pediatric asthmatic patients and their caregivers about how to manage asthma and identify severity of asthma attacks​
  • Goal of bettering knowledge, control, quality of life, fewer hospitalization, etc.​

Installation

  1. Clone GitHub Here
  2. Use OIT Server to run*
  3. Use text editor of choice to modify code and test

      4. To run in VS Code

    1. cd TaBProject/
    2. run npm i

      3. To run website

      1. cd TaBProject/frontend
      2. run npm i

      To run on OIT server

      1. execute docker compose up
      2. If changes to source code occur, run docker compose up -build

      To run mobile app

      1. cd TaBProject/mobile/tab-to-stop
      2. run npm install expo
      3. execute npx start -tunnel, press y

General Features

  • Front Page
    1. Navigate to site

    Results: User can see front page with Login, Register, Resources, and About US buttons.

  • Resources
    1. Click "Resources" button on Front Page before loggin in or registering

    Results: Can view and click on links to various resources.

  • About Us
    1. Click "About Us" button on Front Page before logging in or registering

    Results: Can view information about project sponsor and other owners.

  • Navbar
    1. Login doctor or patient
    2. Navbar is visible on every page in top right corner

    Results: User can utilize Home, Profile, Information, and Logout icons.

  • Back Button
    1. Login doctor or patient
    2. Back Button is visible in top left corner on every page
    3. Also visible from Front Page after clicking buttons

    Results: Back Button navigates to previous page.

Doctor Features

  • Doctor Login
    1. Navigate to the front page
    2. Click "Login"
    3. Click "Doctor"
    4. Enter correct email and password
    5. Click "Submit"

    Results: Doctor is logged in and routed to the Doctor home page. Here they can see a list of assigned patients.

  • Doctor Registration
    1. Navigate to the front page
    2. Click "Register"
    3. Click "Doctor"
    4. Enter requested information
    5. Click "Submit"

    Results: Doctor is regitered and routed to the Doctor login page. Here they can login.

  • Doctor View Profile
    1. Doctor login
    2. Click profile icon on navigation bar

    Results: Doctor can view their profile with their information.

  • Doctor Edit Profile
    1. Navigate to Doctor Profile
    2. Click "Edit Profile"
    3. Enter new information
    4. Click "Save"

    Results: Doctor can alter profile information as needed.

  • See Assigned Patients
    1. Doctor login
    2. View assigned patient list on home page

    Results:Doctor can only see those patients that have registered with corresponding doctor email.

  • View Individual Patient Info
    1. Doctor login
    2. Click "Patient Info" next to desired patient

    Results: Doctor can see patient action sheet

  • Change Patient Medication
    1. Doctor login
    2. Click "Change Medication" next to desired patient

    Results: Doctor can edit patient medication which is reflected on patient action sheet.

  • Archive Patient
    1. Doctor login
    2. Click trash can icon next to desired patient

    Doctor can remove patient from assigned patient list without deleting patient account.

  • Doctor Logout
    1. Doctor login
    2. Click logout icon in navigation bar

    Results: Doctor is returned to Home Page

Patient Features

FAQs

  • Q: How do I download my Action Sheet?

    • A: Login to patient account, click "Action Sheet", click "Download"

  • Q: How do I access the Animations?

    • A: Login to patient account, click "Animations", select animation to watch

  • Q: How do I change my patient's medication?

    • A: Login to Doctor account, click "Patient Info" next to desired patient, click "Change Medication". From here you can add, edit, and delete patient medication.

  • Q: How do I edit my profile?

    • A: Login to Doctor or Patient account, click profile icon in navigation bar, click "Edit Profile". From here you can edit any profile field and save the changes.
